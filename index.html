Sobre mim

Olá! Sou Danilo, um desenvolvedor fullstack apaixonado por criar soluções tecnológicas eficientes e visualmente atraentes. Minha jornada na programação começou há alguns anos e desde então venho aprimorando minhas habilidades para entregar produtos de qualidade.

Trabalho com uma variedade de tecnologias, mas meu foco principal está em Java, Node.js, React e todo o ecossistema ao redor dessas ferramentas. Essa combinação me permite desenvolver aplicações robustas, escaláveis e com excelente experiência de usuário.

Tenho orgulho de ter desenvolvido projetos como Startink e TrêsD Online, que demonstram minha capacidade de criar soluções completas, do backend ao frontend.

Algumas tecnologias com as quais trabalho atualmente:

  • Spring Boot
  • Node.js
  • Express
  • React
  • JavaScript
  • HTML & CSS
  • SQL & NoSQL
